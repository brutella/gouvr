# GOUVR

This project provides a boilerplate to read data from the data bus from devices manufactured by TA (Technische Alternative)

- UVR 31  
- UVR 42  
- UVR 64  
- HRZ 65  
- EEG 30  
- TFM 66  
- [UVR 1611][UVR1611-Website] available under `uvr/1611`
- UVR 61-3
- ESR 21
